Connect to Firebase

One-time setup — about 5 minutes. You'll never have to do this again.

  1. Open Firebase Console and sign in with Google
  2. Click Add Project → name it (e.g. "goff-family") → skip Analytics
  3. In your project: Build → Firestore Database → Create database → start in test mode
  4. Then: Build → Authentication → Get started → enable Anonymous
  5. Click the gear icon → Project Settings → scroll to "Your apps" → click </> Web icon → register app
  6. Copy the entire firebaseConfig object and paste it below
